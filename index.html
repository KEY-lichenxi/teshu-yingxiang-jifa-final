📋 作业说明

本次期末作业旨在全面评估学生在 DaVinci Resolve 软件中的综合能力，涵盖前期创新、后期制作、编剧编写三大核心领域。学生可选择以下三个项目中的至少一个完成，鼓励有能力的学生完成全部三个项目以获得更高评价。

每个项目都设计了不同的技能培养重点，确保学生在学期结束时能够独立完成后期的完整工作流程。

1

Fusion 特效制作项目

核心技术 · Fusion页面 · 3D合成

📝 项目主题

使用 DaVinci Resolve 的 Fusion 页面，制作一条完整的特效视频内容。

题材不限（科幻、广告、MV、实拍合成等均可），但核心特效必须在 Fusion 页面中完成。

⚙️ 技术要求（必须满足）

本作业为 Fusion 综合能力考核，请务必满足以下所有要求：

使用 Fusion 页面完成核心制作

  • ✅ 核心动画、特效、合成必须在 Fusion 页面完成
  • 鼓励使用节点式工作流进行复杂合成

Fusion 关键帧动画（Keyframe Animation）

  • 必须使用 Keyframe（关键帧）控制至少以下内容之一：
    • 📍 位移（Position / Translation）
    • 📏 缩放（Scale）
    • 🔄 旋转（Rotation）
    • 👁 透明度（Opacity）
  • 🌟 鼓励使用 Spline 编辑器优化动画节奏（缓入 / 缓出）

Fusion 3D 系统

  • 作品中必须使用 Fusion 3D 工作流，包括但不限于：
    • ✔ 3D 空间（3D Workspace）
    • ✔ Camera3D / Light
    • ✔ Text3D 或导入的 3D 模型（FBX / OBJ 等）
    • ✔ 2D转3D（ImagePlane3D）
  • 💡 免费 3D 模型可在 Sketchfab 下载

纯原创视频创作项目

原创能力 · 版权意识 · 综合制作

📝 项目主题

制作一条完全原创的视频内容，从拍摄到后期全部由学生独立完成。

本项目重点培养学生的原创意识和版权合规意识，确保学生在未来工作中能够合法合规地进行视频创作。

🎬 内容要求

视频内容必须为纯自创

  • ✅ 自己拍摄的视频内容（旅拍等自主创新拍摄内容）
  • ✅ 自己制作的动漫（并非只有一帧，需为动态作品）
  • ✅ 使用无版权音乐（Creative Commons / Public Domain）
  • ✅ 自己制作的 AI 生成视频：AI 电影、AI 微电影、AI 动漫等（需注明使用的 AI 工具）
  • ✅ 自己录制的环境音、配音等

🎵 推荐的 AI 创作工具（不限于此）：

即梦 · HappyHorse · Gemini · Midjourney · 小云雀 · LibTV 等
AI 工具使用无限制，可自由选择适合自己创作需求的工具

🚫 版权禁令（非常重要）

以下内容严格禁止

  • ❌ 使用别人的游戏画面进行剪辑
  • ❌ 使用别人的动漫进行剪辑
  • ❌ 使用别人的视频（包括电影、电视剧、综艺、短视频等）
  • ❌ 使用未经授权的商业音乐
  • ❌ 使用网络图片/视频素材而未获得明确授权

✅ 推荐的无版权资源网站

  • 🎵 音乐： Creative Commons · Free Music Archive · Incompetech
  • 🎥 视频素材： Pexels Videos · Pixabay Videos
  • 🖼 图片素材： Unsplash · Pixabay

⚙️ 技术要求

  • 使用 DaVinci Resolve 进行完整后期制作（剪辑、调色、音频处理）
  • 鼓励使用调色页面进行色彩校正和风格化调色
  • 鼓励使用 Fairlight 页面进行音频后期处理
  • 成片时长：1-30 分钟（可根据内容适当调整）
  • 分辨率：1920×1080 或以上

旁白创作与音频编辑项目

故事理解 · 音频编辑 · 双语字幕

📝 项目主题

基于老师提供的"天门祭" 剪辑视频，进行旁白创作、录制与字幕制作。

🎥

天门祭 参考视频

https://www.youtube.com/watch?v=HbcH5197UNE

本项目旨在培养学生的故事理解能力、编剧编写能力、音频编辑能力，以及双语字幕制作能力。

📋 项目流程

📖

第一步

了解背景故事

✍️

第二步

撰写旁白文稿

🎙

第三步

录制/生成旁白

📝

第四步

制作双语字幕

📖 具体任务要求

了解背景故事

  • 仔细观看老师提供的"天门祭"视频
  • 理解视频的文化背景、故事情节、情感基调
  • 进行必要的资料查阅，确保对内容有深入理解

旁白文稿创作（故事总结能力）

  • 根据视频内容，撰写完整旁白文稿
  • 旁白应能够：
    • 介绍视频的背景和文化意义
    • 对视频中的关键画面进行说明和解读
    • 总结视频想要传达的核心主题
  • 文稿长度应与视频时长匹配，避免过长或过短

旁白录制（音频编辑能力）

  • 语言要求：使用 英文 或 日文 进行旁白录制
  • 录制方式（二选一）：
    • 🤖 使用 AI 语音生成（需注明使用的 AI 工具）
    • 🎙 自己进行录音（需保证音质清晰）
  • 使用 DaVinci Resolve 的 Fairlight 页面进行音频编辑：
    • 调整音量平衡
    • 添加必要的音频特效（降噪、均衡器等）
    • 与背景音乐/环境音进行混音

字幕制作（双语能力）

  • 字幕语言规则：
    • 如果旁白是日文 → 添加 英文字幕
    • 如果旁白是英文 → 添加 日文字幕
  • 使用 DaVinci Resolve 的字幕功能或Fusion 标题工具制作字幕
  • 字幕要求：
    • 时间轴准确，与旁白完全同步
    • 字体清晰易读，大小适中
    • 位置合理，不遮挡画面重要内容

⚙️ 技术要求

  • 使用 DaVinci Resolve 完成全部音频编辑和字幕制作
  • 鼓励使用关键帧控制音频音量和字幕动画
  • 成片时长：与原始视频保持一致（或根据旁白需要进行适当调整）
  • 输出格式：MP4（含字幕烧录） 或 MOV（含字幕轨道）
